Informations

Bonjour, I am Michel (michel.c at gmail) and I created Solitaire-Play in August 2014. Twenty years ago, I had already written a few games with Visual Basic 3. The very first was Golf, a solitaire card game with no drag & drop to manage. Today, with HTML, CSS and JavaScript I should be able to do it again.

Credits

  • CSS: I made it simple and start with a simple guide to responsive design by Adam Kaplan to build a 7 columns grid.
  • JavaScript: I keep JQuery for basic stuff and jQuery UI for drag & drop. I also give a try to jQuery UI Touch Punch by Dave Furfero to hack a touch event support.
  • Cards: I use Consolas font, Vectorized Playing Cards by Chris Aguilar, and card back from SVG-cards by David Bellot. The final sprite is compressed with TinyPNG.
